  Check with the ECE260B course staff for the correct node.

NOTE: The `prep` command sources ACMS_MODULES and sets up the full EDA environment.
This cannot be done via non-interactive SSH exec_command (used by this agent)
because `prep` relies on interactive shell initialization.
